# Test-Frontend-Tunaiku-Angular
Front end Test for Tunaiku ( Angular 4 Version )

## Installation ##

Make sure yo've already installed Node > v6

1. Go to terminal
2. Type " npm install "
3. Done.
4. To serve on your local machine, go to your terminal and type " ng serve "
( Default port :4200)
